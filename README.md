# Marketing Lens

Marketing Lens is a scalable, intelligent, and cost-effective marketing analytics platform designed for the Microsoft Azure ecosystem.

## Architecture

The project is structured into three main components:

- **frontend/**: A multi-tenant dashboard built with [Astro](https://astro.build) and [React](https://reactjs.org).
- **backend/**: Data ingestion and processing logic using [Azure Functions](https://azure.microsoft.com/en-us/services/functions/) (Python).
- **database/**: SQL scripts and schema definitions for [Azure Database for PostgreSQL](https://azure.microsoft.com/en-us/services/postgresql/).
